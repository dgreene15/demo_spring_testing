Mocking HTTP Requests
*WireMockServer
    - com.github.tomakehurst.wiremock-jre8
    - more complex
    - offers advanced features like request verification, recording, playback, scenario-based testing
*MockWebServer
    - com.squareup.okhttp3.mockwebserver
    - simpler and good for basic needs to simulate HTTP server behavior
